Sistema em desenvolvimento para a disciplina de Prática de Desenvolvimento de Software - PIDS do curso de bacharelado de Sistemas de Informação - SI da Universidade Estadual de Goiás - UEG. 

Este projeto tem como objetivo auxiliar aos alunos do curso de história do Campus de Ciências Sócio-Economicas e Humanas - CSEH de Anápolis-GO no cadastro de atividades complementares.

Nome do sistema: Sistema de Lançamento de Atividades Complementares - SLAC.
Desenvolvido por: Lisianthus Technology.
